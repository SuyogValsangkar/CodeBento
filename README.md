# CodeBento
Compact in form. Simple in design. Rich in content.

CodeBento - A lightweight, clean, powerful web notebook for writing and running code instantly.

## Architecture
- Frontend: React + Vite
- Backend API: Fastify
- Execution Runner: WebAssembly
- Orchestration: SpinKube (K8s)
